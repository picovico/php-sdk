##Functions Overview
* `login()`

Validating user with username and password

* `set_token()`

Set the request token in the project for Session management and authentication

* `start_project()`

Start a new project and receive a new video id for the project

* `open_project()`

Open the project for adding and editing it

* `add_image()`

Uploading images form external source

* `get_library_image()`

Get the list of uploaded images

* `add_library_image()`

Add the iimages form already added music

* `add_text()`

Adding the text frame into project

* `add_music()`

Adding music from external source

* `get_library_music()`

Get the list of added / library music on Picovico 

* `add_library_music()`

Add the music to library of Picovico

* `get_style()`

Get the available styles in Picovico.

* `set_style()`

Set available in library

* `set_quality()`

Set the quality of output video

* `add_credit()`

Add the credits name on the last of video clip

* `get_video()`

get the status of current video

* `save_video()`

Save th information of video in server

* `create_video()`

Start rendering video for creating
